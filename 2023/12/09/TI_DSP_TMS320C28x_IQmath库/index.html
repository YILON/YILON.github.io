概述

德州仪器 TMS320C28x IQmath 库是一套高度优化的高精度数学函数库，
供C/C++程序员在TMS320C28x器件上将浮点算法无缝移植到定点代码中。
这些例程通常用于计算密集型实时应用，在这些应用中，最佳执行速度和高精度至关重要。
通过使用这些例程，可以获得比用标准ANSIC语言编写的同等代码快得多的执行速度。

IQmath库包含五部分：

  • IQmath头文件
    • C 程序使用IQmathLib.h
    • C++ 程序使用IQmathLib.h和IQmathCPP.h
  • IQmath库
    该库包含所有 IQmath 函数和查找表。该库有两个版本:
    • IQmath.lib:
      这是一个索引库，根据 CCS 10.x 中的选项，项目中将使用相应的 EABI 或 COFF 库。
      该索引库使用IQmath_coff.lib和IQmath_eabi.lib库创建。
    • IQmath_f32.lib:
      这是一个索引库，根据 CCS 10.x 中的选项，项目中将使用相应的 EABI 或 COFF 库。
      该库可与使用 --float_support=fpu32 构建的代码链接。
      该索引库使用 IQmath_fpu32_coff.lib 和 IQmath_fpu32_eabi.lib 库创建。
    • IQmath_coff.lib：
      这是 IQmath 库的 COFF 版本。该库专为在C28x上运行。构建该库时未使用fpu_support = fpu32。
    • IQmath_eabi.lib：
      这是IQmath库的EABI版本。该库专为在C28x上运行。构建该库时未使用-fpu_support = fpu32。
    • IQmath_fpu32_coff.lib：
      这是IQmath库的 COFF 版本。该库用于在C28x+FPU设备上运行。该库使用 -fpu_support = fpu32 构建。
    • IQmath_fpu32_eabi.lib：
      这是IQmath库的EABI版本。该库用于在C28x+FPU设备上运行。该库使用 -fpu_support = fpu32 构建。
  • IQmath.cmd
    链该文件分配了 IQmath 库使用的部分.
    dsp中某些段的位置由设备指定.
    例如，IQsin 和 IQcos 功能使用的表格位于设备的Boot ROM 中。
  • GEL 文件
    这些gel函数在使用不直接支持 IQ 数据类型的 Code Composer Studio 版本时非常有用。
    使用 Code Composer Studio V3.3 或 Code Composer Studio V4 时不需要 .gel 文件。

安装

IQmath库包含在Ti公司提供的ControlSUITE套件。

controlSUITE 不再更新，但仍可继续下载。
C2000的最新软件版本现在通过 C2000Ware 提供，并通过 DigitalPower SDK 和 MotorControl SDK 提供应用示例。

安装C2000ware后，
默认在C:\ti\c2000Ware_X_XX_XX_XX\libraries\math\IQmath\c28处为IQmath库。

将 IQmath 运算转换为浮点运算

  • 在IQmath头文件启用FLOAT_MATH，这将转换所有IQmath函数为等效的浮点型函数。
  • 将浮点数写入设备寄存器时，需要将浮点数转换为整数。
    同样，从寄存器读取数值时，也需要将其转换为浮点数。
    在这两种情况下，都需要将数值乘以一个转换系数。
