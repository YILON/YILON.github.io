TI_DSP_TMS320F2812_AD转换单元

发表于2023-11-04|更新于2023-11-23|嵌入式DSP
|浏览量:

概述


ADC模块有以下特点：

  • 12bit精度；(由于结果寄存器是16bit，因此需要将结果左移4位才能得到正确结果)
  • 2个采样和保持器；（采样保持采集时间窗口有独立的预定标控制）
  • 同时或顺序采样模式；
  • 模拟输入电压范围0~3V；
  • ADC时钟最高可配置为25MHz，即最高采样带宽为12.5MSPS；
  • 16通道模拟输入；
  • 自动排序功能支持16通道独立循环“自动转换”，每次转换的通道可以软件选择；
  • 排序器可以工作在2个独立的8通道排序器模式，也可以工作在16通道级联模式；
  • 排序器可工作在启动/停止模式，允许“多个排序触发”同步转换；
  • F2810/F2811/F2812的B版本有增强的重叠排序器功能
  • AD转换公式为D=Vin−ADCLO3×(212−1)\rm{D = \frac{V_{in}-ADCLO}{3} \times (2^12 - 1)}D=3Vin​−ADCLO​×(212−1)
  • 多个触发源：
    • S/W 软件立即启动
    • EVA、EVB 事件管理器启动
    • 外部引脚
  • 运行每个或每隔一个序列转换结束后产生中断请求

注意，为减小数字信号转换引起的噪声对ADC产生耦合干扰，
需要将ADC模块的电源输入与数字电源隔离开。

排序器操作方式与转换模式

排序器是管理ADC的各个通道的采样优先级的。 （按优先级使用采样/保持器）
每有一个设定的触发信号进来，
排序器按设定的优先级按顺序进行一轮采样。
因此可通过设计触发信号的频率来设定ADC采样的频率。

ADC模块的排序器由2个8状态排序器（SEQ1、2）构成（双排序器方式），
二者分别对应8个16bit的结果寄存器RESULTx。
二者可级联构成一个16状态的排序器SEQ（单排序器方式、级联操作方式）。

排序器的状态指排序器内能够完成的AD自动转换通道的个数。

而ADC转换也有同时与顺序两种转换模式。
前者是输入采样保持器A、B各取一通道的信号并行转换，
后者则是同一时刻只有一个通道可以进行转换。

可以设置对同一通道多次采样，提高采样精度。

排序器操作方式

级联操作方式

在启动ADC之前，
需要初始化进行转换的通道数MAX_CONVx，
配置需要的转换输入信号对应的转换次序CONVxx，
最终转换结果存放到对应的结果寄存器RESULTx。

其中最高位确定采用哪一个排序器，后3位定义偏移量

双排序器操作方式


可见此时EVA可以触发SEQ1，
EVB可以触发SEQ2.

双排序器顺序采样

每个排序器的转换个数受ADCMAXCONV寄存器的一个值MAX_CONVn控制。
（此时ADCMAXCONV寄存器的被分为两部分的MAX_CONVn）
当转换开始时该值被装载到自动排序状态寄存器AUTO_SEQ_SR的排序计数控制位SEQ CNTR[3~0]，每经过一轮转换SEQ CNTR[3~0]减一，直至0停止转换。

一旦SEQ CNTR[3~0]递减至0：

  • 当CONT_RUN = 1
    转换序列重新开始，MAX_CONVn重新装载到SEQ CNTR[3~0]中；
    为避免覆盖先前转换的结果，必须保证在下一个转换序列开始之前读取结果寄存器的值。
  • 当CONT_RUN = 0
    排序指针停留在最后状态，SEQ CNTR[3~0]保持为0。
    为了在下一个启动时重复排序操作，在下一个触发信号到来之前必须使用ADCTRL2寄存器的RST SEQn位复位排序器。

每次SEQ CNTR[3~0]归零时，都会置位INT SEQn，
因此可以在该中断的服务子程序中对排序器进行复位。

例如要SEQ1完成ADCINA2与ADCINA3、ADCINA2、ADCINA3、ADCINA6、ADCINA7、ADCINB4的转换。（某些输入重复采样）

双排序器同时采样

类似于级联排序同时采样，
此模式两排序器分别有4个转换次序可用：
CONV00~CONV03与CONV08~CONV11
可用来分别设置4对相同偏移量的通道。

注意上面的转换次序CONVxx分别对应两个排序器SEQ1、SEQ2，
ADCMAXCONV寄存器的取值参见寄存器说明。

排序器的启动/停止模式(CONT RUN)

上面提到，
对于ADC控制寄存器1ADCTRL1的连续运行控制位CONT RUN，
其能够决定排序器是工作在连续运行模式还是启动/停止模式，
前者指排序器的指针会自动复位，
而后者的指针则不会自动复位。
利用启动/停止模式的特点可以实现多个触发源对应采样通道的有序同步。

以一个例子说明。
现要求对EVA的GP1，
定时器下溢信号（在计数为0时出现）作为触发源1启动对模拟信号I0、I1、I2、I3、I4、I5、I6、I7、I8、I9的采样与转换；
定时器周期匹配信号（在计数值=设定周期值时出现）作为触发源2启动对模拟信号V1、V2、V3的采样与转换。
假设按序采样的时间可忽略不计。

可见触发源1对应的待采样通道多于8个，不能以一个排序器来完成这个任务。
此时可使用级联排序器方式，
比如将I0~I9通道对应放到CONV00~CONV09，
V1~V3对应到CONV10~CONV12
设定CONT RUN = 0
此时：

当GP1计数器为0时，ADC模块对I0~I9进行采样，
采样结束后，排序器指针没有归位，仍停留在CONV09；
当GP1计数值=设定周期值，继续采样剩下的被设定的通道V1~V3

注意点：

排序器被触发后，采样的通道数由ADCMAXCONV确定，
注意到上面例子相邻两轮采样的通道数是不一样的。
一个方法是每轮采样结束都利用采样结束中断来设置ADCMAXCONV的值。

排序器指针也是，当V1~V3采样完毕，需要将排序器指针复位至CONV00，
这也可以通过中断实现。

输入触发源

具体的，ADC模块的触发源有：

  • 只要排序器处于空闲状态，SOC触发源就能启动一个自动转换排序。
    空闲状态是指：
    在收到触发信号前排序器的指针指向CONV00，
    或者是排序器已经完成了一个转换排序也就是SEQ CNTRn为0时。
  • 如果转换序列正在进行时，到来一个新的SOC触发信号，
    则ADCTRL2寄存器中的SOC SEQn位置位(该位在前一个转换开始时已被清除)。
    但如果又有一个SOC触发信号到来,则该后来信号将被丢失，
    也就是当SOC SEQn位置位时(SOC挂起)，随后的触发不起作用。
  • 被触发后,排序器不能在中途停止或中断。
    程序必须等到一个序列的结束或复位排序器，
    才能使排序器返回到初始空闲状态(SEQ1和级联的排序器指针指向CONV00，SEQ2的指针指向 CONVO8)
  • 当SEQ1/2用于级联同时采样模式时，SEQ2的触发源被忽略，而SEQ1的触发源有效。
    因此级联模式可以看做SEQ1有最多16个转换通道的情况。

ADC参考电压

C281x处理器的ADC单元的参考电压有2种提供方式：内部参考电压和外部参考电压。
具体选择哪种参考电压由控制寄存器3的第8位EXTREF控制。
为了获得良好的增益性能，处理器要求2个参考引脚ADCREFP和ADCREFM的电压差为1V。
一般情况下，
ADCREFP的电压为2×(1±5%)V2\times (1\pm5\%)\rm{V}2×(1±5%)V
ADCREFM的电压为(1±5%)V(1\pm5\%)\rm{V}(1±5%)V
