数据类型

Verilog中共有19中数据类型。

常量

数字

整数

  • 二进制整数(b或B)
  • 八进制整数(o或O)
  • 十进制整数(d或D)
  • 十六进制整数(h或H)

数字的表达方式有三种：

  • <位宽><进制><数字> 比如8’hac ，表示1010_1100
  • <进制><数字> 默认位宽由系统决定，至少32位
  • <数字> 默认为10进制

x值和z值

在数字电路中，x代表不定值，z代表高阻值。

负数也是以补码的形式存储，注意位运算。

网络类型

表示结构体之间的物理连接。
网络类型的变量不能存储值，而且必须受到驱动器（比如门或者连续赋值语句assign）的驱动。
如果没有与驱动器连接，则该变量就是高阻的，即值为z。
常用的网络数据类型有wire与tri，
wire通常用来表示单驱动器驱动的网络型数据；
tri则用来表示多驱动器的网络型数据。

如果这两种类型的变量没有定义逻辑强度(logic strength)，
在多驱动源时逻辑会发送冲突，从而产生不确定值。
比如当两个驱动源共同作用时：

wire/tri 0 1 x z
0 0 x x 0
1 x 1 x 1
x x x x x
z 0 1 x z

wire型数据通常用来以assign关键字指定的组合逻辑信号。
Verilog模块的输出输入信号默认定义为wire型。
wire型数据可以用做任何方程式的输入，
也可以用作assign语句或实例元件的输出。

reg型

寄存器时数据存储单元的抽象，对应数据类型为reg。
通过赋值语句可以改变寄存器存储的值，
其作用和改变触发器存储的值相当。

reg类型变量默认初始值为不定值x。

reg型数据常用来表示always块内的指定信号，常代表触发器。

定义：

reg型变量可赋正值，也可赋负值。
但当其表示一个表达式的操作数时，视作无符号值，
因此要注意负数以补码方式存储。

数组

Verilog数组维数没有限制。
对于多维数组，使用时必需指定具体每一维的索引。

memory型

通过对reg型变量建立数组可对存储器进行建模，用以描述RAM、ROM或reg文件。
这样的数组便称其为memory类型变量。

定义：

一个完整的存储器不能直接赋值，
就像C语言的数组也不能直接整体赋值。

整型integer

整数类型用关键字 integer 来声明。
声明时不用指明位宽，位宽和编译器有关，一般为32 bit。
reg 型变量为无符号数，而 integer 型变量为有符号数。

实数real

实数用关键字 real 来声明，可用十进制或科学计数法来表示。
实数声明不能带有范围，默认值为 0。
如果将一个实数赋值给一个整数，则只有实数的整数部分会赋值给整数。

时间time

Verilog 使用特殊的时间寄存器 time 型变量，对仿真时间进行保存。
其宽度一般为 64 bit，通过调用系统函数 $time 获取当前仿真时间。

字符串

字符串保存在 reg 类型的变量中，每个字符占用一个字节（8bit）。
因此寄存器变量的宽度应该足够大，以保证不会溢出。

字符串不能多行书写，即字符串中不能包含回车符。
如果寄存器变量的宽度大于字符串的大小，则使用 0 来填充左边的空余位；
如果寄存器变量的宽度小于字符串大小，则会截去字符串左边多余的数据。

有一些特殊字符在显示字符串中有特殊意义，例如换行符，制表符等。
如果需要在字符串中显示这些特殊的字符，则需要在前面加前缀转义字符\ 。

运算符

与c语言的运算符是基本相同的。

算数运算符

几个注意的点：

  • 取模，或者叫求余，要求%两侧均为整型数据，结果值的符号位取第一个操作数的符号位。
  • 整数除法运算时，结果略去小数部分。
  • 算数运算操作时如果某操作数有不确定值x，则整个结果也为x。

位运算符：

常用的：

  • 取反（单目运算符）：~
  • 按位与：&
  • 按位或：|
  • 按位异或：^
  • 按位同或：^~

不同长度的数据进行位运算时，会自动将两个右对齐，然后高位填0。

逻辑运算符

  • 逻辑与：&&
  • 逻辑或：||
  • 非：!

&&和||的优先级低于关系运算符，
!高于算数运算符。

关系运算符

<、>、<=、>=

  • 正常情况下按照关系返回0或1。
  • 如果某个操作数的某位不定，则关系是模糊的，返回值为不定值。
  • 关系元素运算符的优先级低于算数运算符。

等式运算符

  • 等：==
  • 不等：!=

当这两个运算符的操作数存在不定值x或高阻值z的位时，返回不定值x。

  • 全等：===
  • 不全等：!==

这两个运算符能对操作数的z或x位也进行比较，看其是否完全一致，返回0或1。

四个运算符的优先级相同。

移位运算符

  • 逻辑左移：<<
  • 逻辑右移：>>
  • 算术左移：<<<
  • 算数右移：>>>

算术左移和逻辑左移时，右边低位会补 0；
逻辑右移时，左边高位会补 0；而算术右移时，左边高位会补充符号位。

条件语句

if-else

注意else总是和最近的if对应，
因此如果要嵌套使用时，注意将成对的if-else放在同一个块内。

case

  • 不同于c语言的switch语句，其不会执行除了匹配常量对应语句外的其它语句。（不需要break关键字）
  • 所有表达式位宽必须相等，特别注意使用常量作为分支时要注明位宽，否则很可能出现位宽不匹配。
  • 常用表达式之间可以不互斥，虽然这些条件选项是并行比较的，但执行效果以谁在先为准。
  • 如果希望多个常量表达式对应同一个待执行语句，可以将常量表达式并列并用逗号分隔；

考虑到不定值x和高阻值z，Verilog提供了另外两种case分支语句：
casez和casex。
使用方法和case相同，区别在于常量表达式匹配的方式。
三者真值表如下（真值为1表示匹配，0表示不匹配）：

下面只注明不匹配的：

1 1

z 1

case虽然语法上可以比较x或z，
但是不能进行综合，因此还是使用casez或casex比较好。

casez 0 1 x z

1 1 1

z 1 1 1 1

casex只注明不匹配的：
