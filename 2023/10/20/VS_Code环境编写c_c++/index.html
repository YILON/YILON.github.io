WIndows环境下VS Code部署C/c++

  • 安装Vscode
  • 下载编译器的压缩包，将解压得到的文件夹直接放进D盘中
  • 快捷键win + q打开搜索框，搜索环境变量，打开“编辑系统环境变量”，点击“环境变量(N)”，双击系统变量的path，添加编译器的bin文件夹(绝对路径)，比如D:\mingw-w64\mingw64\bin。

完成后在cmd控制台输入

即可验证是否添加成功。
