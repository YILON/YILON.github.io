搭建个人博客（三）——博客文章格式

发表于2022-09-29|更新于2023-07-15|博客搭建
|浏览量:

概述

hexo框架支持markdown格式的文章。而博客页面除了文章主题外，一些附加信息比如标题、时间、标签、分类，需要按照一定格式在markdown文件中注明。可参见butterfly主题官方说明。

Forint-matter

在markdown 文件最上方以 — 分隔的区域称为Forint-matter，在此处可以注明文章相关信息。（不同框架、不同主题所需的信息可能不一样）

以下是一些常用的信息：

类别 作用
title 标题
date 文章创建日期
updated 文章更新日期
type 文章类型，一般不需要设置
comments 评论模块开关（默认为开）
description 文章描述，方便搜索引擎链接
keywords 文章关键词，方便搜索引擎链接
top_img 文章顶部图片路径
cover 文章缩略图路径
mathjax mathjax显示开关(当设置mathjax的per_page: false时，才需要配置，默认 false)
katex katex显示开关(当设置katex的per_page: false时，才需要配置，默认 false)
aside 侧边栏显示开关 (默认 true)
aplayer 在需要的页面加载aplayer的js和css
highlight_shrink 代码框展开开关
categories 文章分类
tags 文章标签

还有其它一些信息，比如侧边目录TOC、文章复制版权信息等等，可参见butterfly主题官方说明。
